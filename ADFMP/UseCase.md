# Сценарии использования приложения

Введем следующие определения:

_Домашний экран приложения_ - экран, который видит пользователь при открытии приложения.

_Главный экран приложения_ - экран приложения, который содержит основную механику приложения 
(в данном случае, кнопку для кликанья).

_Навигационные кнопки_ - стандартные сенсорные кнопки для навигации, встроенные в телефон: "Меню", "Домой" и "Назад".

## Начать новую игру

<ins>Действующее лицо</ins>: пользователь.

<ins>Предусловие</ins>:

* Пользователь еще не начал новую игру и находится на домашнем экране приложения.

<ins>Основной сценарий</ins>:

1. Пользователь нажимает на кнопку "Начать игру".
2. Пользователь попадает на экран с созданием новой игры.
3. Пользователь заполняет все необходимые поля:
    * Имя болезни - текстовое поле.
    * Сложность - выбор из списка.
4. Пользователь нажимает на кнопку "Начать игру".

<ins>Альтернативный сценарий</ins>:

* Пользователь вводит уже существующее название болезни.
    * Пользователь получает уведомление с просьбой придумать другое название.
* Отмена операции
    * Пользователь нажимает навигационную кнопку "Назад".
    * Пользователь попадает на домашний экран приложения.

<ins>Результат</ins>:

1. Пользователь создает новую игру.
2. Пользователь попадает на главный экран приложения.
