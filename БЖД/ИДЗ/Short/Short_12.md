## Задание 2. Авария с выбросом хлора

### Характеристика аварии

| Параметр | Значение |
|----------|----------|
| СДЯВ | Хлор, 10 т |
| СВУА | Изотермия |
| Степень химической опасности | III |
| Разряд химической опасности | 3 |

### Параметры зоны поражения

| Параметр | Значение |
|----------|----------|
| Радиус смертельного поражения | 320 м |
| Радиус тяжёлого поражения | 480 м |
| Глубина распространения облака | 2,5 км |
| Ширина зоны заражения | 380 м |
| Высота облака | 20–30 м |
| Площадь возможного заражения | 5 км² |
| Время подхода к университету (1 км) | **5 минут** |
| Время поражающего действия | 4–6 часов |

### Прогноз потерь (при 4 000 чел. в университете)

| Показатель | Значение |
|------------|----------|
| Общие потери | ≈1 300 чел. (33%) |
| Смертельные | ≈325 чел. |
| Тяжёлые | ≈520 чел. |
| Лёгкие и средние | ≈455 чел. |

### Средства защиты

| СИЗ | Назначение |
|-----|------------|
| Противогаз ГП-7 с коробкой «В» | Основное средство |
| Влажная повязка с 2% раствором соды | При отсутствии противогаза |
| Самоспасатель СПИ-20 | Для эвакуации (15–20 мин) |

### Действия по защите

**Немедленно (0–5 мин):**
1. Сигнал тревоги → укрыться в здании
2.  Закрыть окна, двери, отключить вентиляцию
3. Подняться на верхние этажи (хлор тяжелее воздуха!)
4. Надеть противогаз или влажную повязку с содой

**На открытой местности:**
- Двигаться перпендикулярно ветру
- Не спускаться в низины и подвалы
- Закрыть органы дыхания влажной тканью

**Главное правило:** Хлор тяжелее воздуха — **вверх, а не вниз! **
